דוגמאות רלוונטיות

מה שכבר בניתי בעולמות האוטומציה ו-AI

Email + LLM

סינון וניתוב אימיילים עסקיים

מערכת שמקבלת מאות אימיילים ביום, מקטלגת אותם, מחלצת מהם נתונים מובנים (חשבוניות, הזמנות, פניות), ומנתבת לזרימות עבודה שונות. כולל שמירה היסטורית ו-fallback ידני למקרים מורכבים.

Voice + Claude

תמלול וסיכום שיחות מכירה

שיחות נקלטות, מתומללות עם Whisper, מועברות ל-Claude לחילוץ: רגשות, התנגדויות, action items, הזדמנויות. הסיכומים מוזרמים אוטומטית ל-CRM של הלקוח.

PDF + OCR + Claude

חילוץ נתונים מחשבוניות וטפסים

פייפליין שמקבל PDF/תמונה, מזהה את סוג המסמך, מחלץ את כל השדות הרלוונטיים (גם כשהפורמט לא סטנדרטי), מאמת מול מסד נתונים, ומכניס למערכת. שיעור דיוק 95%+.

Multi-system Sync

סנכרון בין מערכות לא תואמות

אינטגרציה דו-כיוונית בין CRM, ERP, ו-Google Sheets ללקוח שעבד עם שלוש מערכות שלא דיברו. אין אובדן נתונים, יש log מלא, וכל ההבדלים מוכרעים לפי כללים שהלקוח הגדיר.

Claude Agents

סוכני AI ליישומים עסקיים

סוכנים שעובדים עם tools (Calendar, Email, DB, API חיצוני), מקבלים החלטות, ופועלים אוטונומית. עם guardrails, audit trail, ו-human-in-the-loop למקרים קריטיים.

Form Automation

מילוי טפסים אוטומטי באתרים חיצוניים

בוטים שעובדים מול אתרי לקוחות (כולל כאלה עם CAPTCHA), ממלאים טפסים מתוך מסד נתונים פנימי, ומדווחים על תוצאות. שילוב של Playwright + Claude להבנת UI דינמי.
